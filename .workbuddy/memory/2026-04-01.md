## 阵容助手油猴脚本修复（第二次）

**问题：** 在 Electron 应用内嵌浏览器中使用油猴脚本时，`PresetTeamService.getInfo` 返回的数据是 BON 编码的二进制数据（Uint8Array），而不是 JavaScript 对象，导致无法正确解析阵容信息。

**原因分析：**
1. Electron 环境中的 API 返回原始二进制数据，需要 BON 解码
2. 原始脚本假设返回的是已解码的 JavaScript 对象

**修复内容：**
1. 在脚本开头添加完整的 BON 解码器（BonDecoder 类）
2. 更新 `getCurrentTeamInfo()` 方法，检测 body 是否为 Uint8Array，如果是则进行 BON 解码
3. 添加调试日志显示 body 类型和解码后的数据
