# 2026-04-05 工作记录

## 修复 PeachInfo.vue 的 dateStr.split is not a function 错误

- **根因**：`a-date-picker` `@change` 回调在用户清空日期时会传入 `null`（不是 `undefined`），导致 `queryDate.value = null`，进而 `formatDateToShort(null)` 里调用 `null.split` 报错。
- **修复**：
  1. `formatDateToShort` 增加类型判断，兼容 Dayjs 对象、Date 对象和 `null/undefined`。
  2. `fetchBattleRecordsByDate` 将 `undefined != val` 改为 `val != null`，并对非字符串的 Dayjs 对象做 `.format()` 转换，清空时 fallback 到 `getLastSunday()`。
